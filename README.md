## Determinant Calculator
*calculate_determinant(list)*
Calculates the determinant of any N * N Matrix.

## Usage
```python
import determinant_calculator

two_dimensional_matrix = [[1,2],[3,4]]
calculate_determinant(two_dimensional_matrix) # returns -2

three_dimensional_matrix = [[1,2,3],[4,5,6],[7,8,9]]
calculate_determinant(three_dimensional_matrix) # returns 0 

four_dimensional_matrix = [[1,3,5,9],[1,3,1,7],[4,3,9,7],[5,2,0,9]]
calculate_determinant(four_dimensional_matrix) # returns -376
```
